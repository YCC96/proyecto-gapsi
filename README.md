# proyecto e-Commerce GAPSI
 
gapsi-front
=====================

Proyecto construido con:
- ReactJs

Comandos para ejecutar el proyecto

```
cd gapsi-front
npm install
npm start
```

node-api
=====================

Proyecto construido con:
- NodeJs

Comandos para ejecutar el proyecto

```
cd node-api
npm install
npm start
```
